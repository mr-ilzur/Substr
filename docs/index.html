Поиск подстроки

Подстрокой — называется любая выбранная часть строки. Например для строки hello можно выделить следующие подстроки: ello, llo, o, el и множество других. Данный интерфейс позволяет возвращать подстроку из переданной строки.
Для того чтобы это сделать необходимо в "Поле ввода данных:" ввести строку, индекс и длину подстроки. Реализацию вы сможете увидеть в окне "Результат".

Результат
